# Student Marks Analyzer (Python)

A simple Python program that collects marks for multiple subjects and calculates basic statistics.

## Features
- Accepts marks for multiple subjects
- Calculates total marks
- Calculates average marks
- Finds maximum and minimum marks

## Technologies Used
- Python 3

## How to Run
1. Download or clone the repository
2. Run the program using:
   python student_marks_analyzer.py

## Sample Output
